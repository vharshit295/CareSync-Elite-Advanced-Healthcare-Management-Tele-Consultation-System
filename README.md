--CareSync Elite
CareSync Elite is an advanced, web-based Healthcare Management and Tele-Consultation System designed to digitize and streamline the clinical operations of modern medical facilities. It aims to bridge the gap between patients, doctors, and administrators through a secure and efficient unified platform.

--Key Features
Smart Queue Management: Intelligent token-based appointment scheduling system with real-time waiting time estimation.

Tele-Consultation: Integrated browser-native video consultation using the Jitsi Meet API, complete with simultaneous AJAX-based real-time chat.

Digital Prescriptions & Records: Secure digital archives for prescriptions and laboratory reports, accessible anytime via personal dashboards.

Anonymous Mode: Privacy-first feature allowing patients to consult for sensitive health issues without revealing their personal identities.

Role-Based Access Control: Isolated dashboards for Patients, Doctors, and Administrators ensuring data security and privacy.

--Technology Stack
Backend: PHP 8.2

Database: MySQL 10.4 (MariaDB)

Frontend: HTML5, CSS3, JavaScript (ES6+), Font Awesome 6.0

Video Integration: Jitsi Meet External API
